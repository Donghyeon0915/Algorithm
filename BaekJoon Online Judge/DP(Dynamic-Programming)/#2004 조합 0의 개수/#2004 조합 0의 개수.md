## 문제
nCm 의 끝자리 의 개수를 출력하는 프로그램을 작성하시오.

## 입력
첫째 줄에 정수 ***n***, ***m*** (0 <= ***m*** <= ***n*** <= ***2,000,000,000***, ***n*** != 0)이 들어온다.

## 출력
첫째 줄에 
nCm 의 끝자리 0의 개수를 출력한다.
